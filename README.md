## Problems Identified
1. Overwhelming navigation and too many choices.
2. Hero section lacked clarity and immediate value.
3. Product section did not explain how the 5 products connect.
4. Trust-building content appeared too late.
5. Too many competing CTAs.

## What I Prioritized
Because of the 4–5 hour time limit, I focused on:
- Simplifying the navigation
- Redesigning the hero section
- Rebuilding the product overview section
- Creating a responsive layout for desktop and mobile

## Design Decisions
- Reduced navigation options to make the first action clearer.
- Reframed the hero around the needs of school principals.
- Turned the product list into an ecosystem-style section.
- Added stronger trust signals near the top of the page.

## If I Had More Time
- Build the full testimonials and FAQ section
- Add micro-interactions and animations
- Create a more detailed product dropdown
- Improve accessibility and motion states
